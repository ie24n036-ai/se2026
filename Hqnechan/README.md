# 🎮 オンラインマルチプレイRPGサーバー

設計書（`design.md`）に基づいた、マルチプレイ風ゲームのバックエンドおよびAPIサーバーのプロジェクトです。

## 🚀 構成
Hqnechan/
├── app.py             # Flask バックエンド（ゲームサーバー本体）
├── design.md          # ゲーム設計書
├── requirements.md    # 要件定義書
└── templates/         # 画面テンプレート
    ├── title.html     # タイトル＆サーバー選択画面
    └── game.html      # メインゲーム＆インベントリ画面

## 🛠 機能（データ同期システム）
- **プレイヤーデータの保存**: ログインした冒険者の名前、HP、インベントリをSQLiteで管理します。
- **リアルタイム座標同期**: ゲーム画面で移動ボタンを押すと、非同期通信（API）でサーバーのデータベースへ即座に座標が保存されます。
- **チャットログ同期**: プレイヤー同士のチャット発言をサーバーに記録し、3秒ごとに自動で画面に同期します。
